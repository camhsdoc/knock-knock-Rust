# knock-knock

A small Rust networking app implementing a simple knock-knock style client/server application.

## Build

cargo build

## Run

cargo run

## Usage

This project contains binaries under `src/bin/`.

Run a specific binary with:

cargo run --bin <name>

List available binaries with:

cargo run --bin

## Notes

Learning project focused on:
- Rust networking
- sockets
- basic client/server design
